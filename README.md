# Poseidon
Reform B2Bi/EDI with Block Chain Technology to build a trust business enviroment

## Goal
From technical aspect view, the enterprise IT systems could be devided into two parts: Enterprise Internal system,such as ERP,CRM,HR and so on; And the data flow among enterprises(supply chain), normally it is called EDI(Electronic Data Interchange)/B2Bi(Business to Business Integration). The B2Bi is based on some protocols, such as AS2(), OFTP(),Rosettanet,etc.
